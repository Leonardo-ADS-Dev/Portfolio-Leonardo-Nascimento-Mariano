# 💼 Portfólio Pessoal - Leonardo

Portfólio pessoal moderno e responsivo desenvolvido para apresentar meus projetos e habilidades como desenvolvedor Full Stack.

## 🚀 Tecnologias Utilizadas

- **HTML5** - Estrutura semântica
- **CSS3** - Estilização moderna com gradientes e animações
- **JavaScript** - Interatividade e animações
- **Font Awesome** - Ícones
- **Google Fonts** - Tipografia (Poppins)

## ✨ Funcionalidades

- ✅ Design responsivo para todos os dispositivos
- ✅ Navegação suave entre seções
- ✅ Animações e efeitos visuais
- ✅ Preview interativo de projetos com iframe
- ✅ Formulário de contato
- ✅ Menu hamburger para mobile
- ✅ Efeitos de hover e transições suaves

## 📂 Estrutura do Projeto

```
portfolio/
│
├── index.html          # Página principal
├── style.css           # Estilos
├── script.js           # Scripts e interatividade
└── README.md           # Documentação
```

## 🌐 Como Usar

1. Clone o repositório:
```bash
git clone https://github.com/Leonardo-ADS-Dev/portfolio.git
```

2. Abra o arquivo `index.html` no navegador

Ou acesse a versão online: [Seu Link Aqui]

## 📝 Personalização

Para personalizar o portfólio:

1. **Informações pessoais**: Edite o `index.html` com seus dados
2. **Projetos**: Substitua os projetos de exemplo pelos seus
3. **Cores**: Modifique as variáveis CSS em `:root` no `style.css`
4. **Formulário**: Configure o Formspree substituindo `YOUR_FORM_ID`
